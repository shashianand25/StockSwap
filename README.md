
  # StockSwap Mobile App Design

  This is a code bundle for StockSwap Mobile App Design. The original project is available at https://www.figma.com/design/lOkBF6CIt21mYgavlylxoj/StockSwap-Mobile-App-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  Common typo: If you type `nmp run dev` you'll see "The term 'nmp' is not recognized" — that's a typo. Use `npm run dev` instead.
  
  If `npm` isn't recognized at all, ensure Node.js is installed and on your PATH:
  - Check Node.js & npm: `node -v` and `npm -v`
  - Install deps: `npm install`
  - Start server: `npm run dev`

  Optional (PowerShell alias): run `Set-Alias nmp npm` to temporarily allow `nmp` as a shortcut in the current session.
