┌─────────────────────────────────────────────────────────────────────────────┐
│ ✅ SUCCESS VS ❌ FAILURE PATHS                                              │
└─────────────────────────────────────────────────────────────────────────────┘

SUCCESS PATH:
  Enter email → Send OTP → Receive email → Enter correct OTP → Login → Dashboard

FAILURE PATHS:

1. Invalid Email:
   Enter email → Send OTP → ERROR: Invalid email format

2. OTP Not Received:
   Enter email → Send OTP → Wait → Check spam → Resend OTP

3. Wrong OTP:
   Enter email → Send OTP → Receive email → Enter wrong OTP → ERROR → Try again

4. Expired OTP:
   Enter email → Send OTP → Wait > 10 min → Enter OTP → ERROR: Expired → Resend

5. Magic Link Instead of OTP:
   Enter email → Send OTP → Receive magic link → ❌ WRONG!
   → FIX: Update Supabase template with {{ .Token }}
